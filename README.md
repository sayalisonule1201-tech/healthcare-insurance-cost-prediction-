# Healthcare Insurance Prediction System

A Streamlit-based machine learning application that predicts healthcare insurance cost based on user inputs like age, BMI, smoking habits, and other health factors.

---

## 🚀 Live Demo
https://your-streamlit-link.streamlit.app/

---

## ✨ Features
- Predict insurance cost using ML model
- User-friendly Streamlit interface
- Real-time prediction
- Data-driven insights for healthcare planning

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Scikit-learn

---

## ▶️ How to Run
pip install -r requirements.txt  
streamlit run app.py
